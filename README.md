LifeHub is a full-stack AI health technology platform designed to analyze global economic and health data. It provides real-time life expectancy predictions and extracts intelligent, context-aware insights from datasets using Machine Learning and advanced LLMs.

---

## ✨ Key Features

- **🤖 Machine Learning Predictions:** Input current health/economic metrics to get precise life expectancy predictions powered by a trained Scikit-Learn model.
- **📊 Interactive Dashboard:** Explore global data through dynamic line charts, bar charts, scatter plots, and an interactive world map.
- **🧠 AI Chart Analysis:** Click the "✨ Analyze" button on any chart to get instant, real-time insights generated by Google's Gemini 3.1 Flash Lite based on the specific filtered data.
- **💬 Smart AI Chatbot:** An integrated chatbot ready to answer user queries and provide further analysis of the dataset.
- **📱 Fully Responsive:** Seamless experience across desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack & Architecture

### **Frontend**
- **React.js (Vite):** Fast and modern UI development.
- **Recharts & React Simple Maps:** For rendering dynamic, responsive data visualizations.
- **Tailwind CSS / Custom CSS:** For a sleek, modern, and fully responsive glass-morphism design.
- **Deployment:** Vercel

### **Backend & Machine Learning**
- **Python & FastAPI:** Robust API routing and handling requests.
- **Pandas:** For data manipulation and cleaning.
- **Scikit-Learn:** For the predictive ML model.
- **Google Gemini API (3.1 Flash Lite):** For intelligent chart analysis and chatbot capabilities.
- **Deployment:** Hugging Face Spaces

### **Dataset**
- **Source:** Global health and economic data sourced from **IEEE**.
- **Metrics included:** GDP, Infant Deaths, Alcohol Consumption, Vaccination Coverage (Hepatitis B, Polio, Diphtheria), BMI, HIV/AIDS prevalence, and more.
